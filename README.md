## About the Game

Julia's Cycle is a small non-profit fan game inspired by "The Coffin of Andy and Leyley", developed by a solo developer using Godot 4 and GDScript.

The project was created with the intention of telling a story while serving as a first step into the world of game development.

The game is available on Itch.io.

---

## Features

- Dynamic dialogue system using Dialogic 2
- Interactive phone call system
- New characters and original story
- Inventory mechanics
- Original hand-made CGs (No AI used)
- Additions to the headcanon while respecting the events and lore established in the original story (currently based on the pre-burial Chapter 3 timeline)

---

## Technologies

- Godot Engine 4
- GDScript
- Dialogic 2
